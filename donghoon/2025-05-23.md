## ✅ 구현 완료 사항

### 📌 1. 게시글 리스트 페이징 처리
- 게시판 리스트에 페이징 기능 구현
  - 한 페이지당 게시글 수 지정
  - 페이지 번호를 기반으로 리스트 출력
  - 이전 / 다음 페이지 이동 기능 포함

### 📌 2. 게시글 검색 기능
- **카테고리 + 제목 검색 기능** 구현
  - 카테고리 선택 시 해당 게시글만 필터링
  - 제목 일부 단어 포함 시에도 검색 가능 (`LIKE` 검색)
- 검색 결과에 대해서도 페이징 처리 적용

### 📌 3. 게시글 조회수 증가 기능
- 게시글 상세 보기 시 조회수 1 증가
  - 동일 사용자의 반복 조회에 대한 중복 방지 로직 포함

### 📌 4. Git 반영 사항
- 모든 작업은 **`Dong` 브랜치**에 `push` 완료
